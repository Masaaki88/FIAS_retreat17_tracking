# Real-Time Headtracking with Haar Cascades 

## Introduction

Contribution to the FIAS retreat 2017.

## Usage

| Key      | Description            |
|----------|------------------------|
| s        | Toggle audio           |
| c        | Toggle colormap        |
| f        | Toggle facetracking    |
| r        | Toggle eye replacement |
| o        | Show/Hide options      |

## Dependencies

* Python 2
  * Threading
  * Multiprocessing
* OpenCV
* Numpy
* Matplotlib
* Brian
